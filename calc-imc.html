Calculadora de IMC

Ferramenta responsiva que calcula o Índice de Massa Corporal com base nos dados fornecidos pelo usuário (peso e altura). Utiliza fórmulas padrão de IMC e classifica automaticamente o resultado em categorias como baixo peso, normal, sobrepeso ou obesidade, com destaque visual baseado na faixa alcançada.
Feito com HTML, CSS e JavaScript

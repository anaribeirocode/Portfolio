Quiz Interativo

Sistema de perguntas e respostas com lógica de navegação entre etapas, verificação automática de acertos e exibição de pontuação final. As perguntas são armazenadas em um array de objetos e renderizadas dinamicamente. O usuário recebe feedback instantâneo após cada resposta e o layout é totalmente responsivo.
Feito com HTML, CSS e JavaScript
